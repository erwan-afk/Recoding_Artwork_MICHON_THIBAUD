ReadMe - Dessin d'un sapin de Noël avec p5.js
Qu'avez-vous produit ?
J'ai produit un code JavaScript utilisant la librairie p5.js qui dessine un sapin de Noël avec des ellipses pour représenter les boules de Noël.

Comment décrirais-tu ton premier expérience de programmation autonome?
C'était très excitant d'avoir la possibilité de créer quelque chose de concret à partir de zéro. 
Au début, j'ai dû lire et comprendre la documentation de la librairie p5.js pour savoir comment dessiner des formes simples telles que des rectangles et des triangles. 
J'ai également dû comprendre comment utiliser des variables et les fonctions setup() et draw() pour créer un animation.

Une fois que j'ai compris les bases, j'ai commencé à écrire le code en suivant les instructions données. 
J'ai eu à faire quelques modifications pour ajuster la taille et la position du sapin et les boules de Noël à mes besoins.

En fin de compte, j'ai été en mesure de produire un code fonctionnel qui dessine un sapin de Noël avec des boules de Noël en utilisant p5.js.

Comment le processus de codage est-il différent ou similaire à la lecture et à l'écriture de textes?
Le processus de codage est très différent de la lecture et de l'écriture de textes. 
La programmation requiert une pensée logique et structurée pour écrire des instructions claires et précises pour le programme à suivre.

Cependant, il y a aussi des similitudes entre la lecture et l'écriture de code et de textes. 
Les deux nécessitent de comprendre les conventions de la langue et la grammaire pour produire quelque chose de cohérent et compréhensible. 
De plus, les erreurs peuvent se produire dans les deux cas, nécessitant une correction pour obtenir le résultat souhaité.

Que signifie le code et la programmation pour toi et comment la lecture assignée t'aide-t-elle à y réfléchir davantage?
Pour moi, le code et la programmation signifient la capacité de créer des choses utiles et intéressantes à partir de zéro.
